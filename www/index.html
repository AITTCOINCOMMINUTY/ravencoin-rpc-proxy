Sir, what about perfomance?

  1. Most requests are cached for the lifespan of the current block.
  2. All requests are queued.

    The queue acts as a buffer between the public API endpoint and the Raven core node in order to smooth intermittent heavy loads that can cause Raven to fail.
    This minimize the impact of peaks in demand on availability and responsiveness.
